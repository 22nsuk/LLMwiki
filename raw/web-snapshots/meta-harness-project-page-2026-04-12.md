# Snapshot: Meta-Harness project page
- Capture date: 2026-04-12
- URL: https://yoonholee.com/meta-harness/

## Key excerpts
- Meta-Harness gives the proposer a filesystem containing the full source code, scores, and execution traces of prior candidates.
- The project page contrasts this with prior optimizers that compress feedback into summaries, scalar scores, or short windows.
- It reports up to 10M tokens of diagnostic context per step, versus much smaller budgets for surveyed prior methods.
- Text-classification result highlighted on the project page: 48.6% vs ACE's 40.9%, using much less context.
- The project page also highlights a TerminalBench-2 search trace where the proposer diagnoses confounded prompt edits and pivots to safer additive modifications.

## Why this snapshot exists
This snapshot preserves the trace-first, filesystem-mediated diagnosis pattern beyond the PDF abstract alone.
