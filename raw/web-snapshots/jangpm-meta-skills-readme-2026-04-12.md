## Key excerpts
- The repository offers four meta-skills for Claude Code and Codex: `blueprint`, `deep-dive`, `reflect`, and `autoresearch`.
- Recommended workflow: `blueprint -> deep-dive -> implement -> autoresearch -> reflect`.
- `blueprint` produces system design documents with validation scripts.
- `deep-dive` performs structured, multi-step requirement interviews.
- `autoresearch` optimizes skills via repeated execution, evaluation, prompt mutation, results.json, changelog.md, and a live dashboard.
- `reflect` summarizes sessions, updates documentation targets, and proposes next actions.
- The repo treats skills as reusable folders deployable to both Claude Code and Codex.
